FreeSide is a student run project at the University of Hull (http://www.hull.ac.uk/) which offers free (in both free beer and free speech terms) Unix, Open and Alternative computing services to the student population of the University.
We offer:

  • SSH access to Linux Servers
  • Web hosting with popular scripting languages
  • SFTP file hosting
  • X-Forwarding of popular UNIX applications
  • IRC servers
  • MySQL databases
  • Virtual machines for development
